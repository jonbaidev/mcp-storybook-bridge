API
5 tools included

Everything Claude needs to work with your component library.

list_components
Lists all stories from the Storybook index with optional keyword filter.
get_component_story
Returns full story definition — args, argTypes, tags, and import path.
generate_variant
Generates a CSF3 TypeScript story from a natural language description.
audit_accessibility
Runs a11y audit via addon-a11y and returns violations with fix suggestions.
search_components
Fuzzy-searches components by name, tag, or description across the index.
Setup
Ready in 2 minutes

Works with Claude Desktop and Claude Code out of the box.

1

Install the package

Install globally via npm.

npm install -g @jonbai/mcp-storybook-bridge
2

Add to Claude Desktop config

Edit claude_desktop_config.json and add the server block:

{
  "mcpServers": {
    "storybook": {
      "command": "npx",
      "args": ["-y", "@jonbai/mcp-storybook-bridge"],
      "env": {
        "STORYBOOK_URL": "http://localhost:6006",
        "ANTHROPIC_API_KEY": "sk-ant-..."
      }
    }
  }
}
3

Start Storybook and restart Claude

Run your Storybook on port 6006, then restart Claude Desktop. The 5 tools appear automatically.

4

Or use with Claude Code

claude mcp add storybook npx @jonbai/mcp-storybook-bridge \
  -e STORYBOOK_URL=http://localhost:6006 \
  -e ANTHROPIC_API_KEY=sk-ant-...
